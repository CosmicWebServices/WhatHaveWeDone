Why Did We Make This Documentation

What is the purpose of this?
  • This is a way for everyone on ScratchAT to collaborate to make an amazing project using the most basic node.js/socket.io server ever


    • how does it work?
  • The program works by people editing index.html
  • Whenever it is edited, users will recieve the new file when they reload
  • The server will just relay any socket emitted message to everyone


    • What are the rules?
  • Be nice, this is supposed to be a collaboration, not a "Who Can Redirect To The Most Viruses"
  • Work together, we have a gitter https://gitter.im/Scratch-AT/WhatHaveWeDone
  • Dont commit without testing/asking others to review your code, this is a collab, not your own project
